# アイロンビーズ図案メーカー / Fuse Bead Pattern Maker

画像をアイロンビーズの図案に自動変換する無料Webツール。
https://zerobuildone.github.io/beads-pattern-maker/

- 変換はすべてブラウザ内で完結（画像はアップロードされません）
- 色データ: [maxcleme/beadcolors](https://github.com/maxcleme/beadcolors) (MIT License) をもとに再計算・拡張
- 各ブランド名は各社の商標です。本サイトは各メーカーとは無関係の非公式ツールです
- 購入リンクにはアフィリエイト広告を含みます
